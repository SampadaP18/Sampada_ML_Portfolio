## Favorite Project
**Project 7 — Natural Language Processing** stood out the most because it brought together techniques from the entire course: preprocessing, classification, and feature representation and applied them to human language. Seeing how raw text gets transformed into something a model can learn from made the connection between classical ML and modern AI tools like ChatGPT feel concrete and logical.
